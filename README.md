# Linear Algebra Solutions (Gilbert Strang, 5th Edition)

本项目旨在提供 Gilbert Strang 教授经典教材《线性代数》（第五版）的详尽手写题解。作为计算机科学专业的学习者，深感数学根基对算法理解的重要性，故将个人推导过程开源，以供交流与参考。

## 项目特色
- **过程完整**：还原矩阵运算与逻辑推导的每一处细节，拒绝跳步。
- **高清记录**：利用 Notein 记录，保持数学演算的可读性。
- **持续维护**：随学习进度同步更新。
